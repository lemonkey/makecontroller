Detailed Description

The FastTimer subsystem provides a high resolution timer in a microsecond context.

The longest delays are therefore 2^32 / 1000000 = 4294s.

Initializes a fast timer entry structure.

The event is signified by a callback to the function provided, after the interval specified. The specified ID is passed back to the function to permit one function to work for many events. Pass repeat = true to make the event continue to create callbacks until it is canceled. Note that the timer entry structure needs to be created and managed by the caller.

Parameters:
fastTimerEntry pointer to the FastTimerEntry to be intialized.
timerCallback pointer to the callback function. The function must be of the form void callback( int id ).
id An integer specifying the ID the callback function is to be provided with.
timeUs The time in microseconds desired for the callback.
repeat Set whether the timer repeats or is a one-time event.
