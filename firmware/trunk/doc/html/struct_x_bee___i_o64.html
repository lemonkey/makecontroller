Detailed Description

An incoming packet with IO data from a 64-bit address.

  • source is the 64-bit (8-byte) address of the sender.
  • rssi is the signal strength of the received message.
  • options - bit 1 is Address Broadcast, bit 2 is PAN broadcast. Other bits reserved.
  • samples - the number of samples in this packet.
  • channelIndicators - bit mask indicating which channels have been sampled.
  • data is a buffer that holds the IO values as indicated by channelIndicators.
