Detailed Description

The System subsystem monitors and controls several aspects of the system.

From OSC this subsystem can be addressed as "system". It has the following properties:

  • active (R/W) - not currently meaningful
  • freememory (R) - returns the remaining free memory size
  • samba (W) - requests the board erase itself and return to SAMBA state
  • reset (W) - requests the board reboot
  • serialnumber (R/W) - permits the reading and writing of the serial number
  • buildnumber (R) - permits the reading of the board's build number
