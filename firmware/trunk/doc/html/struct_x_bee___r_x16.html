Detailed Description

An incoming packet with a 16-bit address.

  • source is the 16-bit (2-byte) address of the sender.
  • rssi is the signal strength of the received message.
  • options - bit 1 is Address Broadcast, bit 2 is PAN broadcast. Other bits reserved.
  • data is a buffer that holds the value of the incoming packet.
