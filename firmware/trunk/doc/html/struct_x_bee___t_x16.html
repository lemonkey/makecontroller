Detailed Description

The structure used to transmit an XBee packet with a 16-bit destination address.

  • frameID is the ID for this packet that subsequent response/status messages can refer to.
  • destination is the 16-bit (2-byte) address of the destination.
  • options - 0x01 (disable ACK) or 0x04 (Send with Broadcast PAN ID).
  • data is a buffer that holds the value of the outgoing packet.
