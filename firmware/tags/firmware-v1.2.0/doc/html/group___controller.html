Detailed Description

Functions available to the Make Controller Board.

The Controller module provides an API to devices that are all included on the Make Controller, no matter what interface electronics are used around it. The AppBoard module extends the Controller module to include the devices found on the Make Application Board. Make Controller Kit

The Make Controller Kit is an open source project maintained by MakingThings.
MakingThings code is released under the Apache 2.0 license.
Bug tracker, development wiki and status can be found at http://dev.makingthings.com.
