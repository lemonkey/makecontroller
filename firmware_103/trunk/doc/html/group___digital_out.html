The Digital Out subsystem sends digital values out the 8 high current outputs - 1 (on) or 0 (0ff).

int DigitalOut_SetActive (  int  index,
int  state
) 
 

Sets whether the specified Digital Out is active.

This is automatically called, setting the channel active, the first time DigitalOut_SetValue() is called.
However, the channel must be explicitly set to inactive in order for any other devices to access the I/O lines.

Parameters:
index An integer specifying which Digital Out (0-7).
state An integer specifying the state - on (1) or off (0).
Returns:
Zero on success.
